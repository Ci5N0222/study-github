editor에 git 기능이 내장되어 있다.

## `git diff`
- 최근 commit한 파일과 현재 파일의 차이점을 비교한다.
- j/k 키로 스크롤 이동, q키로 종료

### `git diff 커밋ID`
- 현재 파일과 커밋ID의 차이점을 비교한다.

### `git diff 커밋ID1 커밋ID2`
- 커밋ID1과 커밋ID2의 차이점을 비교한다.

## `git difftool`
- 최근 commit한 파일과 현재 파일의 차이점을 Vim 에디터로 확인할 수있다.
- 위와 동일한 커밋 ID로 비교 가능

** 터미널의 한계로 인해 잘 사용하지 않는 기능 **

Extensions에서 git 관련 라이브러리를 사용할 수 있다.
